# Codice di Calibrazione

This project uses Python with an isolated virtual environment and pinned dependencies.

## Setup Instructions

### 1. Create and activate a virtual environment

Python 3.11.9

From the project root (`CODICE DI CALIBRAZIONE`):

#### Activate in bash
python -m venv .venv
source .venv/bin/activate

#### Activate in powershell
python -m venv .venv
.venv\Scripts\Activate.ps1

#### Upgrade pip
python -m pip install --upgrade pip

#### install requirements
pip install -r requirements.txt
